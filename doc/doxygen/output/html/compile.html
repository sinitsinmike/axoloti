Compiling from source

Windows

Important Regular users are recommended to download an install a release version, rather than compiling from source. Releases are published at https://github.com/axoloti/axoloti/releases.

Install MSYS2

Download and install msys2-i686 from http://msys2.github.io/. Use the default settings. Follow the instructions on the webpage.

Install JDK

Download and run the JDK installer from http://www.oracle.com/technetwork/java/javase/downloads/index.html pick "jdk-8u...-windows-i586.exe". For developers, I recommend downloading and installing the Netbeans with JDK package from http://www.oracle.com/technetwork/java/javase/downloads/jdk-netbeans-jsp-142931.html

Get Axoloti from Github repository

Clone the axoloti repository using the github windows client, or using the command line (the MSYS2 shell provides git command line tools).

1 C:\Users\username> git clone https://github.com/axoloti/axoloti.git
2 C:\Users\username> cd axoloti
3 C:\Users\username\axoloti> git submodule update --init --recursive

Compiling

Open a terminal and enter:

1 C:\Users\username\axoloti> cd platform_win
2 C:\Users\username\axoloti> build.bat
3 C:\Users\username\axoloti> cd ..

Launching

Open, if not opened, a terminal, go to the axoloti directory and run

1 C:\Users\username\axoloti> Axoloti.bat

OSX

Important Regular users are recommended to download an install a release version, rather than compiling from source. Releases are published at https://github.com/axoloti/axoloti/releases.

Install Java JDK

Install XCode

Get Axoloti from Github repository

Install the GitHub application https://mac.github.com/. Or if you have git installed, open a terminal, and enter

1 ~$> git clone https://github.com/axoloti/axoloti.git
2 ~$> cd axoloti
3 ~$> git submodule update --init --recursive

Compiling

Open a terminal, go to the axoloti directory and run

1 ~$> ant

If you don't have ant installed, this can be obtained through "brew"

1 ~$> brew update
2 ~$> brew install ant

that would require one to have http://brew.sh/ installed of course...

Open a terminal, go to the axoloti directory and run

1 ~$> cd platform_osx
2 ~$> ./build.sh
3 ~$> cd ..

Root privileges are required to add a udev rule to grant access to Axoloti Core on USB. The install script will ask you for this.

Launching

Open, if not opened, a terminal, go to the axoloti directory and run

1 ~$> ./Axoloti.sh
